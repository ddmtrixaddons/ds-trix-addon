DS Trix Addon - Google Sheets Addon

Overview

The DS Trix addon is an open source addon for Google Sheets that allows users to download and sync reports from DoubleClick Search on a schedule. The original source code is available at https://github.com/google/ds-trix-addon. The published addon uses this code as is without any changes.

Questions/Problems?

For any issues, please raise a bug tracker on the original github project or raise a post on the public forum. Note that support provided is on a best effort basis.
